\begin{DoxyNote}{Note}
The user should make sure that P\+LL bypass is active, by calling the function\+: {\itshape pac5xxx\+\_\+sys\+\_\+pll\+\_\+bypass\+\_\+enable()} before calling this function.

There is a mandatory 500us delay after the P\+LL is enabled before it is stable and ready to use. This function will enforce the passed in delay after it enables the P\+LL before it returns control to the user. At that point, the user may remove the P\+LL bypass by calling {\itshape pac5xxx\+\_\+sys\+\_\+pll\+\_\+bypass\+\_\+disable()}. This time delay is enforced by polling the Sys\+Tick timer. 
\end{DoxyNote}
