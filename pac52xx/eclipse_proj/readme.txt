The PAC FOC project does not ship with a Release folder, so before building the project, it's important 
to Refresh the project so that make files and dependencies can be regenerated.

In addition, the PAC FOC project uses virtual Folder and File links that are relative to the project folder.
See the "Adding Files and Folders to the project" section of the ".resources\ide_support\eclipse\PAC Eclipse Usage.pdf" 
document for additional information on how to add new files or folders/subfolders.
